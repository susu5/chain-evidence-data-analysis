# SUSU5 Chain Evidence Data Analysis 🧩📊

> SUSU5 链路证据数据分析法 — 将任意数据或文件转化为可决策的证据。

---

## 📖 技能描述 / Skill Description

### 中文

**SUSU5 链路证据数据分析法**是一套结构化的数据与文件分析工作流，强调**先出计划、等待确认、再执行分析**的强制分步流程。适用于任何需要从数据中提取结论、证据、风险和对策的场景。

**适用场景：**
- 分析 Excel / CSV / JSON / PDF / HTML 报告 / 聊天记录 / 系统导出报告
- 销售、教学、运营、财务等多领域业务数据分析
- 数据体检、质量诊断、异常排查
- 对比分析、归因分析、分段对比
- 产出决策建议、SOP、策略方案

**核心方法论：**

```
文件识别 → 分析规划（需用户确认）→ 数据质检 → 字段语义 → 源权威性
→ 业务链路定位 → 统计验证 → 逻辑归因 → 证据分级 → 策略输出
```

### English

**SUSU5 Chain Evidence Data Analysis** is a structured data and file analysis workflow with a mandatory plan-first, confirm-before-execute design. It turns any data source into decision-ready evidence.

**Core Method:**

```
file identification → analysis plan (wait for confirmation) → data quality check
→ field semantics → source authority → business chain placement
→ statistical validation → logical attribution → evidence grading → strategy output
```
